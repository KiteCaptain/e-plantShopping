# e-plantShopping

e-plantShopping is a plant shopping web application built with React and Vite. The project presents Paradise Nursery as an online storefront where users can browse plant categories, learn about the nursery, and move through a shopping experience backed by Redux state management.

## Project Overview

This application is designed as a simple e-commerce style experience for plant lovers. It includes a landing page, an About Us section for Paradise Nursery, a product browsing area, and cart-related components connected to a Redux store.

## Features

- Landing page for Paradise Nursery
- About Us section with a brief nursery introduction
- Product catalog organized by plant categories
- Component-based React interface
- Redux Toolkit store prepared for cart state management
- Vite development and build setup

## Tech Stack

- React
- Vite
- Redux Toolkit
- React Redux
- JavaScript
- CSS

## Getting Started

1. Install dependencies with `pnpm install`
2. Start the development server with `pnpm dev`
3. Build the project with `pnpm build`
4. Preview the production build with `pnpm preview`

## Notes

The cart interface and Redux structure are present in the codebase. Some cart action logic is still scaffolded, so this README describes the current project accurately without overstating checkout functionality.
